# Quickstart (Phase 1)

## Prerequisites

- Install the extension from the .vsix or Marketplace.
- Open a workspace with .das files and a keymap.yaml.
- Configure the output path in settings.

## Build Hotkey File

1. Open the Command Palette.
2. Run `DAS: Build Hotkey File`.
3. If prompted, confirm overwrite of the existing output file.
4. Confirm the success notification and output path.

## Validate Configuration

1. Change a documented setting.
2. Run the build command again.
3. Confirm the new setting takes effect without restarting VS Code.
