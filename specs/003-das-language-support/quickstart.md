# Quickstart (Phase 3)

## Prerequisites

- Install the extension from the .vsix or Marketplace.
- Open a workspace that contains one or more .das files.

## Verify Language Mode

1. Open a .das file in VS Code.
2. Confirm the language mode indicates DAS (status bar).
3. Confirm syntax highlighting for commands, variables, numbers, strings, and comments.

## Verify Navigation

1. Open the Outline or Go to Symbol.
2. Confirm ExecHotkey-referenced names appear in the list.

## Verify Folding and Brackets

1. Place the cursor on a bracket and confirm matching highlights.
2. Use folding controls on a control flow block and confirm it collapses and expands.

## Non-Destructive Editing Check

1. Save the .das file without formatting.
2. Confirm no whitespace or line changes are introduced.
