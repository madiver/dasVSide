# Quickstart (Phase 0)

## Run in Development Mode

1. Open the project in VS Code.
2. Launch the Extension Development Host.
3. Open the Command Palette and run the placeholder command.
4. Confirm the info toast appears.

## Package the Extension

1. Run the standard VS Code packaging tool to produce a .vsix.
2. Confirm packaging completes without warnings or errors.

## Install the .vsix

1. Create a fresh VS Code user profile.
2. Install the .vsix into that profile.
3. Run the placeholder command and confirm the info toast appears.
